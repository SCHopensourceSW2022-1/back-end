실행방법
 1. 깃허브 백앤드 repository에 test 폴더를 다운받는다.
 2. test 폴더 안에 hello(프론트엔드) hello-backend(백엔드) 폴더가 있다.
 3. 프론트엔드 폴더, 백엔드 폴더에서 각각 npm i 명령을 친다.(npm 명령어를 위해서 node.js가 깔려있어야합니다.)
 4. hello-backend 폴더로 들어가서 'npm start' 명령어를 친다.
 5. localhost:8000 으로 접속해본다.
 
 제가 너무 저만 알아볼 수 있게 만든것 같아서 방법도 간단히 첨부해놓았습니다. 혹시 잘 모르시겠으면 질문해주세요!
